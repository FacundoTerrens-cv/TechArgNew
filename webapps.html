Landing-Page

Una Landing Page (página de aterrizaje) es un sitio web de una sola Pagina. Generalmente tiene un diseño más sencillo con pocos enlaces e informaciones básicas sobre la oferta, además de un formulario para realizar la conversión.

Sistema Booking

Es un sistema de manejo y gestion de turnos o reservas, cuenta con una herramienta de administracion para que la persona a cargo del sistema tenga un completo control de quienes soliciten los turnos

WebApps

Una web app (aplicación web en español). No requiere ninguna instalación. Se puede crear un acceso directo en el escritorio del ordenador o en el inicio del Celular mediante un marcador, las WebApps satisfacen las mismas necesidades que las Aplicaciones Moviles.

App

Las apps son herramientas de software para teléfonos inteligentes y tablets. Se caracterizan por ser útiles, dinámicas y fáciles de instalar y manejar.
